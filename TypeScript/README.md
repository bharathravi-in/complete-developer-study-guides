# TypeScript Advanced — 30-Day Mastery Guide

> **Goal**: Master TypeScript's advanced type system, patterns, and ecosystem for senior-level interviews and production codebases.

## 📋 Study Plan Overview

| Week | Focus Area | Days |
|------|-----------|------|

| **Week 2** | Advanced Types & Patterns | Days 8–14 |
| **Week 3** | Runtime, Decorators & Tooling | Days 15–21 |
| **Week 4** | Architecture, Migration & Interview Prep | Days 22–30 |

## 🎯 Key Learning Outcomes

- Master generics, conditional types, mapped types, template literal types
- Understand variance, covariance, contravariance in TypeScript
- Build type-safe APIs and libraries
- Write advanced utility types from scratch
- Use decorators (Stage 3) and metadata reflection
- Configure complex TypeScript projects (monorepos, project references)
- Pass senior-level TypeScript interview questions
