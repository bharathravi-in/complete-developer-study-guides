# Day 20 - Pythonic Coding & Advanced Features

## Topics Covered
- Map/Filter/Reduce
- functools module
- itertools module
- collections module

## Key Modules
### functools
- `lru_cache` - Memoization
- `partial` - Partial function application
- `reduce` - Reduce iterable to single value

### itertools
- `chain` - Chain iterables
- `combinations` - All combinations
- `permutations` - All permutations
- `groupby` - Group consecutive elements

### collections
- `Counter` - Count elements
- `defaultdict` - Dict with default value
- `namedtuple` - Lightweight objects
- `deque` - Fast double-ended queue
