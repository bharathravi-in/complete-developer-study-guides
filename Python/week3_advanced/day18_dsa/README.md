## Big O Complexity
| Name | Complexity | Example |
|------|------------|---------|
| Constant | O(1) | Array access |
| Logarithmic | O(log n) | Binary search |
| Linear | O(n) | Linear search |
| Linearithmic | O(n log n) | Merge sort |
| Quadratic | O(n²) | Bubble sort |
| Exponential | O(2ⁿ) | Recursive Fibonacci |
