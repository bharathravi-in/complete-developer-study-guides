## Who Is This For?

- **Senior engineers** preparing for FAANG / Big Tech interviews
- **Full-stack developers** looking for structured learning paths
- **Tech leads** brushing up on architecture, system design, and best practices
- **Self-learners** wanting a comprehensive, curated curriculum

## How to Use

Each topic follows a consistent structure:

```
Topic/
├── README.md                    # 30-day study plan & roadmap
├── Week-1-.../                  # Foundational concepts
├── Week-2-.../                  # Intermediate patterns
├── Week-3-.../                  # Advanced techniques
├── Week-4-.../                  # Architecture & interview focus
├── Interview-Prep/              # Curated Q&A for interviews
└── Cheatsheets/                 # Quick-reference sheets
```

**Recommended workflow:**
1. Pick a topic → read the `README.md` for the 30-day plan
2. Study one day file at a time with code examples
3. Use `Interview-Prep/` to test yourself
4. Keep `Cheatsheets/` open during revision
