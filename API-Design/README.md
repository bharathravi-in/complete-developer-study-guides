# API Design & REST — 30-Day Mastery Guide

> **Goal**: Master REST API design, GraphQL, API security, versioning, and documentation for senior-level interviews.

## 📋 Study Plan Overview

| Week | Focus Area | Days |
|------|-----------|------|

| **Week 2** | Advanced REST & GraphQL | Days 8–14 |
| **Week 3** | API Security, Testing & Performance | Days 15–21 |
| **Week 4** | API Architecture, Gateway & Interview | Days 22–30 |
