# 🎯 Dart — 4-Week Mastery Plan

This plan takes you from zero Dart knowledge to senior-engineer interview readiness in 28 days. Starting with core language fundamentals and null safety, you'll progress through object-oriented programming, advanced collections, asynchronous programming, isolates, Dart 3 features (records, patterns, sealed classes), FFI, metaprogramming, Flutter-relevant patterns, and performance optimization — finishing with a full mock interview day. Every day file is self-contained with prose explanations, annotated Dart 3.x code examples, and interview Q&As so you can study any topic independently.

## 🚀 How to Use

**Recommended daily routine** (~60–90 minutes/day):

1. **Read** — Work through the day's Core Concepts section. Don't skim; read every paragraph.
2. **Code** — Type out (don't copy-paste) each code example in DartPad or your local editor. Modify them to test your understanding.
3. **Practice** — Attempt to answer the Interview Questions before reading the answers. Write your answers out.
4. **Review** — Re-read the Key Takeaways. Update your progress tracker (change `⬜` to `✅`).

**Estimated time per day:**
- Week 1–2: ~60 minutes (fundamentals move quickly)
- Week 3–4: ~90 minutes (advanced topics need more experimentation)

**Tips:**
- Use [DartPad](https://dartpad.dev) for quick experiments — no setup required.
- Keep a personal notes file alongside each day for things that surprised you.
- On Day 28, treat the mock interview as a real interview: time yourself and write code without looking things up first.
