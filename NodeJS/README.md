# Node.js — 30 Day Mastery Plan

A comprehensive beginner-to-advanced Node.js curriculum with daily lessons, code examples, 270 interview questions, and hands-on exercises.

## 📅 Schedule

### Week 1: Core Foundations (Days 1-7)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 1 | Architecture & Event Loop | V8, libuv, event loop phases, thread pool |
| 2 | Modules & Packages | CommonJS vs ESM, npm/yarn/pnpm, semver |
| 3 | File System & Streams | fs module, readable/writable/transform streams |
| 4 | Async Patterns | Callbacks, Promises, async/await, concurrency control |
| 5 | HTTP Servers | Core http module, routing, body parsing, HTTPS |
| 6 | Events, Buffers & Process | EventEmitter, Buffer API, process management |
| 7 | Child Processes & Workers | spawn/exec/fork, Worker Threads, Cluster |

### Week 2: Express & APIs (Days 8-14)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 8 | Express Fundamentals | Middleware, routing, error handling |
| 9 | REST API & CRUD | RESTful design, pagination, filtering, validation |
| 10 | Database Integration | MongoDB/Mongoose, PostgreSQL/Knex, migrations |
| 11 | Authentication & JWT | JWT, refresh tokens, OAuth 2.0, RBAC |
| 12 | Middleware & Error Handling | Composable middleware, AsyncLocalStorage |
| 13 | WebSockets & Real-time | Socket.IO, SSE, scaling WebSockets |
| 14 | Testing | Jest, Supertest, mocking, coverage |

### Week 3: Advanced Topics (Days 15-21)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 15 | Caching & Redis | Cache patterns, stampede prevention, invalidation |
| 16 | Message Queues & Jobs | BullMQ, RabbitMQ, scheduling, dead letters |
| 17 | Security | OWASP Top 10, injection prevention, CORS, helmet |
| 18 | Performance & Monitoring | Profiling, event loop monitoring, APM |
| 19 | Microservices | Service design, gRPC, API gateway, service discovery |
| 20 | GraphQL | Apollo Server, DataLoader, auth, performance |
| 21 | Docker & Deployment | Multi-stage builds, CI/CD, PM2, Kubernetes |

### Week 4: Production & Interview (Days 22-30)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 22 | Logging & Observability | Pino, Prometheus, OpenTelemetry, Sentry |
| 23 | TypeScript with Node.js | Typed Express, generics, repository pattern |
| 24 | Rate Limiting & Throttling | Token bucket, sliding window, tiered limits |
| 25 | Serverless & Lambda | AWS Lambda, cold starts, Serverless Framework |
| 26 | Design Patterns | Factory, Strategy, Observer, Circuit Breaker |
| 27 | System Design | URL shortener, chat system, job scheduler |
| 28 | Error Handling & Resilience | Circuit breaker, retry, bulkhead, graceful shutdown |
| 29 | Interview: Core & Intermediate | 14 frequently asked questions with answers |
| 30 | Interview: Advanced & System Design | System design, production scenarios, debugging |

## 📊 Stats
- **30 days** of structured content
- **270 interview questions** (9 per day × 30 days)
- **30 hands-on exercises**
- **Beginner → Advanced** progression

## 🚀 How to Use
1. Study one day's material each day (~1-2 hours)
2. Type out all code examples (don't copy-paste)
3. Complete the hands-on exercise
4. Review interview questions — practice explaining aloud
5. Build the mini-project each weekend combining that week's topics

## 🎯 Prerequisites
- JavaScript fundamentals (ES6+)
- Basic command line usage
- A code editor (VS Code recommended)
- Node.js 20+ installed
